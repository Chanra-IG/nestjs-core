# Installation

## Development

1. Clone repository

  ```shell
  $ git clone https://github.com/Chanra-IG/nestjs-core.git
  ```

2. Go to folder, and copy `.env.example` as `.env`

  ```shell
  $ cd nestjs-core
  $ cp .env.example .env
  ```

3. Fill out your .env credential
4. Install dependency

  ```shell
  $ yarn install
  ```

5. Run application in dev mode

  ```shell
  $ yarn dev
  ```

6. Open [http://localhost:3000]()
